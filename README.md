## Simple NYT Api search
---
### Usage
Enter search phrase, add timeframe and search!
### Setup
```
npm install
```
or
```
yarn install
```
then
```
"scripts": {
  "start": "react-scripts start",
  "build": "react-scripts build",
  "test": "react-scripts test",
  "eject": "react-scripts eject"
```
